# Video to markdown!

This is the official extension of Video to markdown! for Chrome and Firefox.

## Requirements

- Node.js >= 22
- PNPM >= 10

## Getting started

```sh
pnpm i
pnpm build
```

Build process will generate two folder:

1. `./dist-chrome`
2. `./dist-firefox`

And two .zip file:

1. `dist-chrome.zip`
2. `dist-firefox.zip`
